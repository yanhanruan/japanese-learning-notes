平台安全架构

**ユーザーログインの流れ**

1. **ユーザーログイン**：ユーザーはアカウントとパスワードを入力し、reCAPTCHA 認証を完了してリクエストを送信する。
2. **データの暗号化と送信**：フロントエンドはハッシュアルゴリズムでデータを暗号化し、Axios を使ってバックエンドに送信する。
3. **バックエンド認証**：Express はリクエストを処理し、認証コードとアカウント情報を確認する。成功した場合、Authorization キーを生成する。
4. **キーの保存**：Authorization キーは localStorage または sessionStorage に保存され、今後の API リクエストに使用される。
5. **セキュリティ対策**：バックエンドはプロキシサーバーを通じてリクエストを処理し、SQL インジェクションや悪意のあるクローリングを防ぐ。

 **〜あまり（に）**

这是一个 **表示因程度过高而导致某种结果** 的表达方式，常译为 “由于太……”、“因为过于……” 或 “过度……结果导致……”。

------

 **结构：**

- **名词 + の + あまり**
   例：心配のあまり → 因为太担心
- **动词普通形 + あまり**
   例：驚くあまり → 因为太吃惊
- **な形容词 + な + あまり**
   例：真面目なあまり → 因为太认真
- **い形容词（原形）+ あまり**
   例：嬉しさのあまり → 因为太高兴

## 通用句式

**一、 開始（开始话题或回答问题时）**

1. はい、その点につきましては、〜
   - **Romaji:** Hai, sono ten ni tsukimashite wa, ...
   - **中文意思：** 是的，关于那一点，... (用于针对面试官提出的具体某点开始回答)
2. 〜についてですね。
   - **Romaji:** ~ ni tsuite desu ne.
   - **中文意思：** 是关于...的事情对吧。(用于确认问题，并自然过渡到回答)
3. まず、〜からお話しさせていただきます。
   - **Romaji:** Mazu, ~ kara ohanashi sasete itadakimasu.
   - **中文意思：** 首先，请允许我从...开始说起。(用于有条理地开始陈述，特别是当需要分点说明时)
4. ご質問いただいた〜につきましては、
   - **Romaji:** Goshitsumon itadaita ~ ni tsukimashite wa,
   - **中文意思：** 关于您刚才问到的...， (非常礼貌地回应特定问题)

**二、 転折（转换话题、补充信息、表达不同方面时）**

1. また、〜
   - **中文意思：** 另外，... (用于补充说明，添加新的信息点)
2. さらに、〜
   - **中文意思：** 此外，更进一步来说... (用于在已有信息上递进补充)
3. 一方で、〜
   - **中文意思：** 另一方面，... (用于从不同角度或对比说明)
4. （少し話は変わりますが、）その点に関連して、〜
   - **中文意思：** （虽然稍微有点跑题）关于这一点，相关的... (用于引入与刚才话题略有不同但相关的点)
5. 加えて、〜
   - **Romaji:** Kuwaete, ...
   - **中文意思：** 加上，而且... (用于添加并列信息)

**三、 まとめ（总结陈述时）**

1. 以上をまとめますと、〜
   - **中文意思：** 总结以上所述，... (用于对前面说过的内容进行概括)
2. このように、〜と考えております。
   - **中文意思：** 因此/像这样，我认为... (用于根据前面的论述得出结论或看法)
3. 従いまして、〜
   - **Romaji:** Shitagaimashite, ...
   - **中文意思：** 因此，所以... (比较正式的因果连接词，用于总结)
4. 〜については以上です。
   - **中文意思：** 关于...就说到这里。 (用于结束对某个特定话题的陈述)

**四、 質問（向面试官提问时）**

1. いくつか質問させていただいてもよろしいでしょうか。
   - **Romaji:** Ikutsuka shitsumon sasete itadaitemo yoroshii deshou ka.
   - **中文意思：** 请问我可以问几个问题吗？ (在面试官允许提问时，先请求许可)
2. 〜について、もう少し詳しく教えていただけますでしょうか。
   - **Romaji:** ~ ni tsuite, mou sukoshi kuwashiku oshiete itadakemasu deshou ka.
   - **中文意思：** 关于...，您能再稍微详细地说明一下吗？ (请求更详细的信息)
3. 〜とは、具体的にどのようなことでしょうか。
   - **Romaji:** ~ to wa, gutaiteki ni dono youna koto deshou ka.
   - **中文意思：** 您提到的...，具体是指什么样的事情呢？ (请求具体的解释)

**其他常用辅助句式：**

- **はい、承知いたしました。** (Hai, shouchi itashimashita.) - 好的，我明白了/知道了。(表示理解或收到指示)
- **恐れ入りますが、もう一度お願いできますでしょうか。** (Osoreirimasu ga, mou ichido onegai dekimasu deshou ka.) - 不好意思，能麻烦您再说一遍吗？(没听清时)
- **そうですね、少し考えさせていただけますでしょうか。** (Sou desu ne, sukoshi kangaesasete itadakemasu deshou ka.) - 嗯...，能让我稍微考虑一下吗？(需要短暂思考时)
